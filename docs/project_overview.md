# 项目概述

## 项目简介

QKD Coexistence Experiment（QKD 共纤传输实验系统）是一款用于自动化控制量子密钥分发（QKD）与经典 DWDM 光信号共纤传输实验的控制系统。

## 项目定位

**核心定位：设备自动化控制系统**

本项目的重点在于自动化控制 QKD 共纤传输系统中的各实验装置（包括 WSS、TLS、OSA、ASE 源等），实现实验流程的自动化运行与数据采集。

仿真模块（`simulation/`）仅作为辅助工具，用于在设计实验方案前评估信道分配方案是否能达到预期效果。

## 研究背景

量子密钥分发（QKD）是一种基于量子力学原理实现安全密钥分发的技术。在实际光纤网络中，如何让 QKD 信号与现有的经典 DWDM 通信信号在同一光纤中共存而不产生过度干扰，是一个重要的研究课题。

本系统通过自动化控制实验装置，研究不同信道分配方案对 QKD 系统性能的影响（以 SKR 安全密钥率和 QBER 量子误码率为指标）。

        ┌───────────────────┼───────────────────┐
        ▼                   ▼                   ▼
┌───────────────┐   ┌───────────────┐   ┌───────────────┐

## 主要功能

1. **设备自动化控制**：通过串口/TCP/IP 控制 WSS、TLS、OSA、ASE 等光学设备
2. **实验流程自动化**：自动执行信道分配方案、采集 QKD 数据、保存实验结果
3. **方案预评估**：使用仿真模块在实验前评估信道分配方案的预期性能
4. **数据分析**：时间序列分析、SKR/QBER 统计
5. **可视化**：通道分配可视化 GUI

## 实验参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 光纤距离 | 10 km | 实验用光纤长度 |
| 经典信号功率 | 13 dBm | 激光源发射功率 |
| QKD 采集时间 | 2400 秒 (40分钟) | 每个方案的数据采集时长 |
| 缓冲时间 | 600 秒 (10分钟) | 设备调整等待时间 |
| 量子频率 | 193.5 THz | QKD 信道频率 |
| 同步频率 | 193.3 THz | 同步信号频率 |
| 信道间隔 | 100 GHz | DWDM 信道间隔 |
| 经典信道数 | 8 | 每组实验中的经典信道数量 |
